# Simple Login Form Project

## Project Overview
This project is a simple login form that allows users to enter their credentials, which can be validated against a set of predefined rules. The form is designed with user experience in mind and incorporates basic validation to ensure correct user input.

## Features
- User-friendly interface
- Basic validation for email and password fields
- Responsive design that works on mobile and desktop devices
- Secure handling of user inputs to prevent security vulnerabilities such as XSS and SQL Injection.

## Technologies Used
- HTML5: For structuring the content
- CSS3: For styling the login form and ensuring a responsive layout
- JavaScript: For client-side validation and enhancements

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/nog19/Login-simples-com-html.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Login-simples-com-html
   ```
3. Open the `index.html` file in your favorite web browser to view the login form.

## Usage
Simply enter your email and password in the respective fields and click on the "Login" button. The form will validate the input and provide feedback based on the validation rules set within the JavaScript code.
